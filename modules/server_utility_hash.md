### hashPassword

::: tip Usage
Athena.utility.hash.**hashPassword**(`plainTextPassword`): `string`
:::

Hash a plain text password with pbkdf2 hash and salt.

Returns a pbkdf2 key, and salt. Which can be seperated by the `$` sign.

#### Example
